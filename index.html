Текст в котором будет осуществляться поиск

Выделение осуществляется с помощью canvas, а не surroundContent стилизованными тегами, чтобы много раз не перестраивать DOM.

Данное решение алгоритмически наивно и служит цели показать наввык владения Range. На самом деле, при добавлении / удалении лишь одной буквы в search / editor, нет необходимости заново искать все вхождения строчки во всем тексте, достаточно модифицировать уже имеющиеся. Для этого можно поддерживать структуру данных вроде префиксного дерева.
